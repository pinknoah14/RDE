## 11. 발견된 운영 인사이트

### ✅ 정상 확인된 항목

1. **파레토 효과 재현**: 상위 5% SKU가 전체 CRITICAL의 90%를 차지 — 실 센터와 동일한 패턴
2. **FEFO 알고리즘**: 혼적 지번 정렬 100% 정확 (45 pair 위반 0건)
3. **today_sales 연동**: 91.5% 후보에 당일 판매량 반영 (v1.9 버그 수정 효과)
4. **matched_bins 노출**: 복수 지번 정보가 검수 화면까지 정상 전달

### ⚠️ 실 운영 투입 전 조치 필요 사항

| 항목 | 내용 | 우선순위 |
|---|---|---|
| proximity 좌표 등록 | `floor_access_points` 테이블에 지게차 통로 좌표 입력 → 🟢/🟠 비율 증가 | 높음 |
| Slack 채널 분리 | SF/SM/PW/NC가 모두 "R존" 채널로 통합 → 존별 채널 분리 설정 | 높음 |
| 보충 불가 78개 처리 | 구매발주 트리거 또는 타 센터 이관 연동 필요 | 중간 (v2.0 고려) |
| SUB 섹션 배정 | 시뮬레이션에서 전체 MAIN → 메자닌(SF/SM) 존 `list_section=SUB` 설정 재확인 | 낮음 |
